# 港股多空分析框架

> 2026-06-28 构建

## 四维交叉验证

```
① 沽空率(15日趋势) → 空头态度(加码/回补/平仓)
② 涡轮Call/Put → 资金方向(抄底/追空/观望)
③ 牛熊证街货 → 散户底线 + 空头目标
④ 缠论信号 → 技术面确认
```

## 信号解读

| 沽空率 | 涡轮Call | 牛熊状态 | 缠论 | 判断 |
|---|---|---|---|---|
| ↘ 持续回落 | 有承接(近平价Call成交) | 牛熊收敛(小间距) | Buy/一买 | 🟢 底部 |
| ↘ 回落 | Call试探(深度价外) | 牛证支撑远 | Sell | 🟡 底部博弈 |
| ↗ 加码 | Put碾压 | 牛证濒死(收回价近) | Sell | 🔴 继续跌 |
| 脉冲回落 | 方向不明 | 牛熊间距大 | 弱 | ⏳ 观望 |

## 关键阈值

```
沽空率>20% + 加速上升 → 不要抄底
沽空率<10% + 持续下降 → 空头撤退, 关注反弹
牛证底线距现价<5% → 定时炸弹(一旦跌破引发踩踏)
熊证活跃区=反弹天花板
Call近平价有成交量 → 有资金接盘
```
