# 港股沽空/窝轮/牛熊证数据格式指南

> 2026-06-28 | 用户粘贴AAStocks TSV数据 → 多空深层分析

## 数据源

用户从AAStocks (aastocks.com) 复制粘贴港股五巨头的沽空/窝轮/牛熊证数据。三个独立数据表，TSV格式（Tab分隔）。

## 1. 沽空数据 (15个交易日)

```
日期	价格	升跌	升跌(%)	沽空股数	沽空金额	成交金额	占大巿沽空比率	沽空比率	3天平均	5天平均
```

**关键字段:**
- `沽空比率` — 当日沽空占成交百分比。**最重要指标**
- `3天平均` vs `5天平均` — 趋势判断：3天<5天=空头回补，3天>5天=加速做空
- `价格` vs `沽空比率` — 价跌+比率升=空头加码，价跌+比率降=空头获利平仓

**解读三层级:**
| 沽空率 | 信号 |
|---|---|
| <10% | 🟢 空头平仓/底部接近 |
| 10-20% | 🟡 中性偏弱 |
| >20% | 🔴 空头加码/远离 |

## 2. 涡轮 Call/Put (窝轮)

```
热门 名称 代号 购/沽 相关资产 发行 现价 升跌 升跌(%) 成交额 溢价(%) 价内/外 实际杠杆 引伸波幅 行使价 街货(%) 街货量 最后交易日
```

**关键字段:**
- `购/沽` — Call还是Put
- `现价` + `升跌(%)` — Put全线上涨=空头主导，Call全线下跌=多头溃败
- `成交额` — 资金流向。**Call高成交=有资金抄底**
- `行使价` — 距现价%。Call行使价距现价<20%=近平价抄底信号；>30%=深度价外"彩票"
- `价内/外` — 正值为价外
- `实际杠杆` — Put>5x=高杠杆空头工具

**多空判断:**
| Call状态 | Put状态 | 信号 |
|---|---|---|
| 近平价Call有成交 | Put温和上涨 | 🟢 资金在抄底 |
| Call全线深度价外 | Put全线上涨 | 🟡 中性偏弱 |
| Call零成交 | Put碾压 | 🔴 无人敢买，远离 |

## 3. 牛熊证 (CBBC)

```
热门 名称 代号 牛/熊 相关资产 发行 现价 升跌 升跌(%) 成交额 溢价(%) 行使价 收回价 杠杆 换股比率 街货(%) 街货量 最后交易日
```

**关键字段:**
- `牛/熊` — 牛证(看涨)还是熊证(看跌)
- `收回价` — **最关键！牛市底线/熊市天花板**
- `街货(%)` — 散户持仓集中度。>30%=高集中，跌破收回价=集体归零=踩踏
- `杠杆` — 熊证杠杆越高→空头越激进
- `距现价%` — `(收回价-现价)/现价×100`。牛证距现价<3%=极度危险

**牛熊底线解读:**
| 牛证距现价 | 信号 |
|---|---|
| <3% | 🔴 定时炸弹 — 跌破=散户割肉潮 |
| 3-8% | 🟡 警戒线 |
| >8% | 🟢 安全距离 |

## 4. 多空综合评分 (四维交叉)

| 维度 | 腾讯示例 | 美团示例 |
|---|---|---|
| 沽空率 | 6.1% 🟢 | 26.2% 🔴 |
| 涡轮Call | 有承接(5830万) 🟢 | 零成交 🔴 |
| 牛熊间距 | 收敛(22%) 🟢 | 混乱(38%) 🔴 |

## 5. 用户粘贴数据处理流程

用户粘贴的TSV数据 → 分析步骤:
1. 识别表格类型（沽空/涡轮/牛熊）— 按标题第一个字段判断
2. 提取关键数值 — 沽空率/行使价/收回价/成交额/街货
3. 趋势判断 — 15日趋势vs最新单日
4. 多空信号汇总 — 四维交叉输出
5. 交叉验证缠论 — 确认/矛盾
