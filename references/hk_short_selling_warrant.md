# 港股沽空+窝轮分析模式

> 2026-06-28 | 会话示例: 美团 (03690.HK)

## 沽空分析 (AAStocks 15日数据)

来源: `https://www.aastocks.com/sc/stocks/analysis/stock-short-selling-ratio.aspx?symbol=03690`

### 趋势判断信号

| 模式 | 含义 |
|---|---|
| 价跌 + 沽空比率升 | 🔴 **空头加码** — 最危险，不能抄底 |
| 价跌 + 沽空比率降 | 🟡 空头回补 — 可能有反弹 |
| 反弹日沽空比率<15% | 空头不留恋反弹，趋势未反转 |
| 沽空3日均 > 5日均 | 空头加速 |
| 沽空比率突然>25% | 🔴 加速做空 — 尖顶信号 |

### 美团6/26实例

```
价格: 79.95 → 64.25 (-19.6% in 15天)
沽空3日均: 16.38% → 21.84% ↑ (加速)
沽空5日均: 17.20% → 19.43% ↑

6/26沽空 26.2% = 15日最高 → 空头在最底部加速做空
唯一弱空日 6/22: 沽空11.2% → 反弹日空头不留恋

→ 结论: 不抄底 = 接飞刀
→ 等沽空比率持续<15% + 价格企稳
```

## 窝轮/牛熊证分析 (AAStocks)

来源: `https://www.aastocks.com/sc/stocks/warrantcbbc/search.aspx?tab=1&symbol=03690`

### 关键指标

| 看什么 | 含义 |
|---|---|
| Put全线上涨 + Call全线下跌 | 🔴 空头绝对主导 |
| 最活跃Put行使价距现价 | 市场在赌跌到多少 |
| Call全是深度价外(>30%) | 抄底Call都是"彩票"，无实值Call成交 |
| Put溢价率 | <10%为机构做空, >20%为散户追空 |

### 美团6/28实例

```
Put全线+8~28%上涨 vs Call全线-2~14%下跌
最活跃Put: 行使价65-75 (距现价+3~17%)
最活跃Call: 行使价83-85 (距现价+30%+) — 全是深度价外

→ 不只机构做空，散户也在买Put
→ 抄底Call是"彩票" — 无实值Call成交
→ 沽空+窝轮双重确认: 不抄底
```

## 与缠论的交叉验证

```
缠论Sell-二卖 + 沽空加速 + Put主导 = 三重确认
→ 缺一反转 → 等缠论Buy信号 + 沽空比率下降
```
