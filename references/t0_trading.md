# 日内做T策略

## 核心限制

chan.py **不支持 30m K 线分析**。CTime 的 `auto=True/False` 均会产生 kline time err。根本原因是 `trigger_load({KL_TYPE.K_DAY: [klu]})` 将多个日内 bar 视为同一天，CTime 拒绝接受。

## 替代方案

不用 30m 缠论，改用：

```
日线缠论定方向(Buy=偏多优先回补, Sell=偏空优先减仓, Hold=中性双向)
  + 30m盘中极值点(今日高低点)
  + MA20偏离(低于MA20 + 放量 >1.3x = 回补信号)
  + 量价异常(缩量 <0.6x = 卖压不足可卖; 放量 >1.5x = 冲高可减仓)
  = 做T信号
```

## 做T纪律

1. **下跌趋势中只卖不买** — 先卖等高回补，严禁加仓摊平
2. **持有底仓才能做T** — A股T+0前提
3. **止损底线不可破** — 日线中枢下沿或250日低点
4. **振幅 <3% 不做T** — 扣掉手续费后无利润空间

## 脚本

```bash
python3 t0_trade.py 002475
# 输出: 今日区间/MA20/ATR/量比/做T建议
```
