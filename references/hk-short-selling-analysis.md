# 港股沽空+涡轮+牛熊证交叉分析框架

> 四维度：沽空率趋势 + 涡轮Call/Put流量 + 牛熊证底线 + 缠论信号

## 一、沽空率分析

数据源：aastocks.com → 个股沽空 → 十五个交易日

关键指标：
- **趋势**：沽空率持续上升=空头加码（最危险）
- **反向信号**：价跌+沽空率降=空头回补（底部接近）
- **陷阱信号**：反弹日+沽空率跳升=空头高位加仓

```
腾讯: 6.1%→15.5% = 反弹被加空 ← 修正底部判断
美团: 26.2%→21.8% = 空头微量回补(太拥挤才退出)
```

## 二、涡轮 Call/Put 分析

看什么：
- Put涨幅 = 空头活跃度
- 最近Call行使价 vs 现价 = 市场预期反弹幅度
- 近平价Call有成交 = 有人抄底（腾讯优于美团的关键差异）

```
美团: Call最近83(+32%外)零成交 → 无人抄底
腾讯: Call 482(+17%外)有5830万成交 → 资金在接
```

## 三、牛熊证底线

- 牛证收回价 = 散户最后的防线（街货集中处）
- 熊证收回价 = 空头定价的反弹天花板
- 牛熊间距越小 = 市场分歧越小（腾讯390-480间距22%最优）

```
美团: 牛63(-2%!) — 致命风险
腾讯: 牛390(-5%) — 安全距离够
阿里: 牛78(-13%) — 风险可控
```

## 四、交叉验证信号排序

```
🥇 沽空率趋势(下降) + Call有承接 + 牛熊收敛 = 最接近底部
❌ 沽空率上升 + Call零成交 + 牛证底线仅2% = 远离
```
