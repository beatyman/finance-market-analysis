# Chanlun Figure Cards

These are original abstract comparison cards for the embedded Chanlun package.
They are portable skill assets, not copied pages from the source PDF.

Use them as visual anchors while reading `../rules.md`.

| File | Use For | Do Not Use For |
|---|---|---|
| `overview.svg` | one-page overview of all cards and usage sequence | detailed rule replacement |
| `fractal_stroke.svg` | top/bottom fractal, inclusion processing, stroke confirmation | automatic K-line recognition |
| `segment_break.svg` | segment composition and segment break reasoning | simple price breakout judgment |
| `center_zg_zd.svg` | center overlap, ZG/ZD, extension/upgrade thinking | exact chart measurement without OHLC data |
| `buy_sell_points.svg` | relative position of first/second/third buy/sell points | declaring a buy/sell point without level and invalidation |
| `divergence_macd.svg` | MACD as auxiliary force comparison | treating MACD as the primary Chanlun rule |
| `interval_nesting.svg` | large-level direction plus small-level timing | mixing unrelated levels |

When live market data is summarized rather than full OHLC sequence, cite the
card as a reference and lower confidence instead of forcing a structure.
