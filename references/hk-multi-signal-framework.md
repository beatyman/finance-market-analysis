# 港股多信号交叉验证框架

> 2026-06-29 建立 | 四维度: 沽空率 + 涡轮 Call/Put + 牛熊证 + 缠论

## 四因子评分体系

### 1. 沽空率 (Short Selling Ratio)
- **数据源**: AASTOCKS 沽空数据 (15个交易日)
- **趋势判断**: 
  - >25% 且上升 = 空头加码 🔴
  - 15-25% 且下降 = 空头回补 🟡
  - <10% 且下降 = 空头撤退 🟢
- **关键**: 日输出必须含 "沽空率 + 5日均 + 趋势方向"

### 2. 涡轮 Call/Put (Warrants)
- **数据源**: AASTOCKS 涡轮数据导出
- **核心指标**:
  - Put 成交量 vs Call 成交量 (Put/Call 比)
  - 最近价 Call 行使价距现价% (越小越好)
  - Call 成交活跃度 (有/无)
- **空头判断**: Put 全线涨 + Call 零成交 = 极端看空
- **底部判断**: 近平价 Call(距<15%) 有成交 → 资金在抄底

### 3. 牛熊证 (CBBC)
- **数据源**: AASTOCKS 牛熊数据导出
- **核心指标**:
  - 牛证街货集中收回价 (散户底线)
  - 熊证活跃收回价 (空头目标)
  - 牛熊间距 (越小 = 分歧越小, 越接近底部)
- **风险**: 牛证收回价一旦触及 → 集体归零 → 加速下跌 (做市商收割)

### 4. 缠论 (Chan Theory)
- **引擎**: chan.py (chan-btc-strategy)
- **等级**: Buy-一买 > Buy-二买 > Hold > Sell-一卖 > Sell-二卖
- **核心**: 必须在中枢内才有安全边际

## 实战案例 (2026-06-29)

### 美团 — 四维全空
| 维度 | 数据 | 信号 |
|---|---|---|
| 沽空率 | 26.2% ↗ 33% | 🔴 空头加码 |
| 涡轮 | Put涨15-33%, Call 0成交, 最近Call+32% | 🔴 无人抄底 |
| 牛熊 | 牛63(濒死), 熊79(活跃), 仅差2% | 🔴 定时炸弹 |
| 缠论 | Sell-二卖, 无中枢 | 🔴 |

→ **结论**: 远离

### 腾讯 — 最接近底部
| 维度 | 数据 | 信号 |
|---|---|---|
| 沽空率 | 17% → 6.1%(15日最低) | 🟢 空头平仓 |
| 涡轮 | Call 482(+17%), 5830万成交 | 🟢 资金在抄底 |
| 牛熊 | 牛390(-5%), 熊480(+17%), 间距22% | 🟢 收敛 |
| 缠论 | Buy-一买, 中枢493-520 | 🟢 |

→ **结论**: 最接近底部, 等回调390-400

### 对比分析口诀
```
沽空率: <10%=多, >25%=空
Call状态: 有成交=多, 零成交=空
牛熊间距: <25%=底部, >35%=分歧
缠论: 中枢内=安全, 中枢外=危险
```

## 操作流程

1. 拉取 AASTOCKS 沽空(15日) + 涡轮 + 牛熊数据
2. 运行 chan.py 日线缠论分析
3. 输出四维对比表 (按买入→规避排序)
4. 给出多空信号结论
