# 港股多空三信号分析框架

## 原理

结合三个独立信号源交叉验证，判断港股底部和反弹预期：

1. **沽空率** — 空头态度（加码/回补/平仓）
2. **涡轮Call/Put** — 资金押注方向（有无人抄底）
3. **牛熊证收回价** — 散户底线 vs 空头目标

## 三信号决策矩阵

| 信号组合 | 沽空率 | Call活跃 | 牛熊收敛 | 判断 |
|----------|--------|----------|----------|------|
| 腾讯模式 | <10% ↘ 平仓 | 有承接 | 间距22% | 🟢 最接近底部 |
| 阿里模式 | <10% ↘ 回补 | 试探 | 间距36% | 🟡 底部博弈 |
| 快手模式 | 15-20% ↗↘ 脉冲 | 无 | 间距42% | 🟡 方向不明 |
| 美团模式 | >20% ↗ 加码 | 零成交 | 间距38%+牛证危 | 🔴 远离 |

## 数据源

- 沽空率: HKEX 日度数据 (aastocks.com: 沽空比率+15日趋势)
- 涡轮: aastocks.com 窝轮牛熊证搜寻 (Call/Put行使价+溢价+成交+街货)
- 牛熊证: aastocks.com 牛熊证 (收回价+街货集中度+距现价%)

## 关键阈值

- 沽空率<10%+连续回补 = 空头投降
- 沽空率>20%+加码 = 空头碾压
- 牛证收回价距现价<5% = 危险（街道集中=做市商有动力压到收回价）
- Call有近平价(<20%价外)+成交>1000万 = 资金在抄底
- 牛熊间距<25% = 市场分歧小(成熟)
- 牛熊间距>35% = 市场分歧大(混乱)

## 操作映射

- 三信号全绿(腾讯模式) → 正股入场 + sell put收权利金
- 两绿一黄(阿里模式) → 轻仓试探
- 一绿两黄(快手模式) → 等
- 三信号全红(美团模式) → 重仓立刻减仓换仓、无仓不要碰
