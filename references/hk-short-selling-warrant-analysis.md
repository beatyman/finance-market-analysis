# 港股沽空+涡轮+牛熊证 多空分析框架

> 2026-06 通过美团/阿里/腾讯/快手四只实战验证

## 三重数据交叉验证

### 1. 沽空率（Short Selling Ratio）

来源：aastocks.com 过往沽空资料

```
关键信号:
  沽空率>20% + 持续上升 = 空头加码，别抄底
  沽空率从高位回落至<10% = 空头平仓，底部接近
  反弹日沽空率跳升(如6→17%) = 空头利用反弹加仓
  反弹日沽空率微降(如26→22%) = 空头微量回补但仍重仓
```

### 2. 涡轮 Call/Put

来源：aastocks.com 窝轮搜寻

```
空头主导信号:
  Put全线大涨 + Call全线大跌 + Call成交量=0
  
抄底信号:
  近平价Call(距<20%)有成交量
  Put跌幅 > Call跌幅
  
深度价外Call(>30%外)有量 ≠ 看多(可能是彩票)
```

### 3. 牛熊证 CBBC

来源：aastocks.com 牛熊证搜寻

```
牛证底线 = 街货最集中的收回价
  跌穿 = 牛证集体归零 = 加速下跌

熊证活跃区 = 成交最大的收回价
  熊证涨得快(杠杆>3x) = 空头在赚钱
  涨得慢(熊证微跌) = 空头在止盈非逼仓

牛熊间距 = |熊收回 - 牛收回|
  窄(<25%) = 市场分歧小，方向明确
  宽(>35%) = 市场分歧大，方向不明
```

## 港股五巨头排序规则

```
1. 沽空率<10% + Call有承接 + 牛熊收敛 → 买
2. 沽空率10-20% + 试探抄底 → 等
3. 沽空率>20% + 无人抄底 + 牛证危险 → 远离
```
